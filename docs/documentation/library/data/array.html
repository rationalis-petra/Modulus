List Seq

Implementation of the [Seq] signature for lists. [Seq] is a generalization [Container] for data-structures where the elements have some kind of ordering.

Monad

Functor

Array is a functor, and so implements map.

Map (binary ¨) 

> map (+ 1) ⦗1 2 3⦘
⦗2 3 4⦘

Applicative

Array is an applicative, and so implements pure and apply. 

> pure 3 
⦗3⦘

> apply ⦗(+1) (-1)⦘ ⦗1 2 3⦘
⦗2 3 4 0 1 2⦘

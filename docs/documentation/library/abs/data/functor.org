* Overview
The Functor is one of the simplest classes: 

#+begin_src modulus
(Functor : (𝒰 → 𝒰) → 𝒰)
(defn Functor [F]
  (signature
    (map : {A B} → F A → (A → B) → F B)))
#+end_src

Note: Functor and Cofunctor are equivalent

** Monad

*** Functor
Array is a =functor=, and so implements =map=.

Map (binary =¨=) 

#+begin_src modulus
> map (+ 1) ⦗1 2 3⦘
⦗2 3 4⦘
#+end_src

*** Applicative
Array is an applicative, and so implements =pure= and =apply=.

#+begin_src modulus
> pure 3 
⦗3⦘
#+end_src

#+begin_src modulus
> apply ⦗(+1) (-1)⦘ ⦗1 2 3⦘
⦗2 3 4 0 1 2⦘
#+end_src
